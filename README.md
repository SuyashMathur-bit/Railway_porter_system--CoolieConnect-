# 🚉 CoolieConnect – Railway Porter Booking System

## 📌 Overview

CoolieConnect is an Android application designed to modernize railway porter (coolie) services by enabling passengers to book porters digitally within railway stations. The application aims to provide a seamless booking experience, fare estimation, and payment management through a user-friendly interface.
